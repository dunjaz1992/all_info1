Линукс команды 
1.pwd (print working directory)> возвращает абсолютный путь до нашей папки
2. ls (list)> выводит список всех файлов и папок текущей директории
3. cd (change directory) <dir_name> перемещает в определенную папку(dir_name)
     cd .. ->перемещаемся на уровень выше (назад)
     cd /home/<username>/dir_name>
     


4 . mkdir <dir_name> -> создание указанной директории
5. clear (ctrl+l) очищает терминал (перемещает)
6.rmdir <dir_name -> удаляет выбранную директории (должна быть пустой)
7. touch <file_name> ->создание файла
8. rm <file_name> удаление файла
9. cat <file_name> выводит содержимое файла
10. nano <file_name> -> открывает/создает файл в текстовом виде
      1.ctrl+X
      2.ctrl+Y
      3.Enter
11. ls -a -> выводит список файлов и директорий, включая скрытые
    ls -la -> тоже самое с подробной информацией 
12. mv <filename> <dir_name> ->перемещает filename в указанную директорию 
    Также служит для переименование файла 
13. cp <filename> <dir_name> -> копирует файл в указанную директорию
14. find <name> -> ищет файл или дирректорию
    find /home -name <filename>
15.sudo <command> -> выполняет команду с правами админа
16. echo <text> > <filename> -> записывает <text> в указанный файл (файл перезаписывается)
17. history -> выводит историю введенных в терминал команд
18. rm -rf <dir_name> -> удаляет все вложенные файлы и папки и только потом удаляет нужную 
(указанную)директорию
